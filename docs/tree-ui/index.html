TreeUI Documentation

The service book for the TreeUI ecosystem. Structured for quick reference and deep production integration.

TreeUI handles menu navigation using a high-performance tree structure. It extends Unity's native parent-child hierarchy to create a modular system with Dictionary-based O(n) traversal.

  • Zero-GC runtime for mobile/VR stability.
  • Integrated Scene Management (Additive Support).
  • Self-Healing Hierarchy Guardrails.

Create the Network

GameObject -> UI -> TreeUI -> Menu Network. This automates the creation of the Manager, Canvas, and Transitions root.

Add Menus

Select a parent and use UI -> TreeUI -> Menu. A setup wizard will manage naming and internal references.

Designer-friendly Animator workflow:

  1. Add a StartTransition Trigger to your Animator.
  2. Call Transition.OpenMenu via an Animation Event on your swap frame.

Access OnMenuOpen and OnMenuClose UnityEvents to trigger logic specifically for individual menus.

MenuManager.cs
